UFO Game

UFO Game is a simple 2d game where you control a UFO and navigating a maze while collecting gold.

UFO Game was the first game I created in unity  and taught me the basics of 2D game programming in Unity

Roll-A-Ball

Roll-A-Ball is a 3D game where you play as a ball navigating 3 Stages for collectibles.

Roll-A-Ball was the first 3D game I created in unity and taught be the basics of 3D programming in Unity such as movement and UI

TANKS!

TANKS! is a more complex 3D multiplayer game where you play as a tank trying to blow up your opponent.

TANKS! was the 2nd 3D game I created in Unity and is more complex than Roll-A-Ball and workd with a lot more code and taught me about particles and lighting

Cityscape run is a 2D endless runner where you ride on a skateboard avoiding obstacles
